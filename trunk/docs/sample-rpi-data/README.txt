You can use this sample data from RPI Scheduler, to attempt to run the Scheduler
as a demo. You might also use it to compare to your own schedule data file.

Files:
- sis-data-200707.txt: sample official RPI course data file
- rpi.schedb: University Scheduler XML file generated from sis-data-200707.txt

- old-satyanash-format.txt: sample RPI data in the old Satyanash Scheduler text
                            format

- rpi.schedui: a University Scheduler UI configuration file for RPI

You may also be interested in the RPI database converter, in the rpidb/ folder
in this distribution, which converts RPI's official text-based course schedule
file into University Scheduler XML format. This may help you to build your own
database converter.
